达拉崩吧

《达拉崩吧》是一首现象级的VOCALOID中文神曲，以其独特的创意和极强的传播力，在二次元文化与主流大众之间架起了一座桥梁。

歌曲简介

这首歌曲由知名音乐人ilem作词并作曲，于2017年3月25日由虚拟歌姬 洛天依 和言和共同演绎后正式发布。歌曲一经推出便在哔哩哔哩等平台引发热潮，播放量增长迅速，最终达成了“VOCALOID中文神话曲”的成就。

故事梗概：歌曲讲述了一位勇者为了解救被巨龙抓走的公主，踏上征途并最终战胜巨龙的故事。

魔性节奏：歌曲副歌部分需要快速、准确地重复这些超长名字，形成了一种饶舌般的快节奏挑战，既魔性洗脑，又因其极高的演唱难度而被戏称为“逼死翻唱之作”。
